# REF

> Print the external reference made within a script.
> >
> Shell - Python - JavaScript - Ruby - Perl

<br>

**Install**
```
git clone https://github.com/NeverWonderLand/REF.git && cd REF && mv ref.py /usr/local/bin/ref && chmod +x /usr/local/bin/ref
```

**Usage**
```
ref example.py
```

**Example**
```
└─# ref spoof.py

Referenced Files:
['spoof.txt']
External Calls:
['scapy', 'ipaddress']
```
